## rubocop

### 修正
修正候補表示
```shell
$ rubocop -a
```
自動修正
```shell
$ rubocop -A
```

### 抑止

#### 全体
```yaml:.rubocop.yml
AllCops:
  NewCops: enable

Style/AsciiComments:
  Enabled: false
```

#### コード内
```ruby 
# rubocop:disable Metrics/AbcSize, Metrics/MethodLength
def foo
  # 長いコード
end
# rubocop:enable all
```

- https://docs.rubocop.org/rubocop/configuration.html
